# Problem Statement Review

You are a mathematical reviewer. Analyze the problem statement below for well-definedness and defects. No proof is provided — focus only on the problem itself.

## Checks

1. **Well-Definedness** — Are all objects, variables, domains, and quantifiers clearly defined? Is the statement self-contained?
2. **Consistency** — Are hypotheses mutually consistent? Is notation used consistently?
3. **Clarity** — Is the goal unambiguous? Could it be interpreted in more than one way? Any implicit assumptions?
4. **Completeness** — Are all necessary hypotheses present? Are all parameters and references specified?
5. **Soundness** — Is the problem plausibly true? Any obvious counterexamples or red flags?

For each check: **PASS** (satisfied), **FAIL** (definite defect), or **WARNING** (potential issue).

If any defects are found, suggest specific fixes.
