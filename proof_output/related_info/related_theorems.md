# Related Theorems

## Directly Applicable Results

**The Fibonacci Recurrence Definition:**
The Fibonacci sequence is uniquely determined by:
- F(1) = 1, F(2) = 1
- F(n) = F(n-1) + F(n-2) for all n >= 3

This definition directly yields F(5) = 5 by iterative computation.

**Binet's Formula (not needed but included for completeness):**
F(n) = (phi^n - psi^n) / sqrt(5), where phi = (1 + sqrt(5))/2 and psi = (1 - sqrt(5))/2.

For n = 5: F(5) = (phi^5 - psi^5) / sqrt(5) = 5.

This formula is overkill for n = 5 but confirms the answer.
