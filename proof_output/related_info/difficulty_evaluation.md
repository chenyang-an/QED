# Difficulty Evaluation

## Classification: Easy

## Justification
This problem asks to compute the 5th Fibonacci number, which is a straightforward arithmetic calculation. The Fibonacci sequence is completely standard: F(1) = 1, F(2) = 1, F(n) = F(n-1) + F(n-2). Computing F(5) requires only three additions: F(3) = 2, F(4) = 3, F(5) = 5. This is elementary arithmetic that any student familiar with the Fibonacci definition can perform in seconds.

## Key Complexity Factors
- No complexity factors present
- Direct computation with no proof required
- No edge cases or subtleties
- The answer is simply the number 5
